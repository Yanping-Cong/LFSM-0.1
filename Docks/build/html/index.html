Reference¶

  • spectral_index – Calculating spectral index with different situation
    • A constant spectral index model
    • A spatial-variance spectral index model
  • interpolate_sky – Using the produced spectral index to produce sky map
    • produce sky map in any frequency
  • fitting_params – fitting parameter of emissivity and produce the smoothed sky map
    • Fitting the parameter of emissivity cylindrical coordinate
    • Integrated the emissivity in each LOS to produce the smoothed sky-map
  • I_E_term – isotropic extragalactic emission
    • The isotropic extragalactic emission except for CMB
  • absorp_sky_map – The ultimate output sky map considering the free-free absorption effect
    • produce absorption included sky map in any specific frequency
